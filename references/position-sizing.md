# Position Sizing

## Objective

Stay alive long enough to capture big winners.

Core rule:

`big winners require concentration, but survival requires risk limits`

## Position Types

### 1. Probe Position

Use when:

- thesis looks promising
- chart is improving but not fully proven
- catalyst is near but not fully confirmed

Typical use:

- smallest starter size
- purpose is information and participation, not maximum profit

### 2. Core Position

Use when:

- thesis is confirmed
- chart quality is strong
- liquidity is sufficient
- invalidation is clear

Typical use:

- largest intended exposure

### 3. Tactical Position

Use when:

- event or squeeze potential is high
- volatility is extreme
- edge is shorter-lived

Typical use:

- smaller than core size
- faster review cadence

## Sizing Principles

- Size by risk, not by excitement.
- The more fragile the thesis, the smaller the size.
- The more violent the chart, the smaller the initial size.
- A clear invalidation level allows larger size than a vague hope trade.

## Suggested Framework

For each trade, define:

- `idea quality`
- `chart quality`
- `liquidity quality`
- `downside if wrong`
- `whether the stock is squeeze-prone`

Then classify:

- `A setup`: can become a core position
- `B setup`: probe first, add only after proof
- `C setup`: watchlist or tactical only

## Practical Rules

- Do not open full size on the first buy unless the user explicitly wants aggressive style.
- Prefer `start -> confirm -> add` sequencing.
- Reduce size when:
  - liquidity is poor
  - the move is mostly narrative
  - binary event risk dominates
  - the stock is already extended

## Concentration Rules

Allow more concentration only when:

- the theme is strong
- the name is clearly the best vehicle
- the catalyst path is visible
- technical structure is high quality

Do not concentrate simply because the stock has gone up recently.

## Risk Questions

Before sizing up, ask:

- If this drops `20%-30%`, is that normal volatility or thesis failure?
- Is the risk a temporary shakeout or permanent impairment?
- Can I add if it works, or am I forcing the entire bet now?

## Review Frequency

- persistent trend leader: slower review cadence
- high-beta momentum trend: active review cadence
- event-driven revaluation: catalyst-based review cadence
- mania / squeeze: intraday or daily review cadence
