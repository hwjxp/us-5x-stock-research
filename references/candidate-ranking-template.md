# Candidate Ranking Template

Use this template when comparing a basket of possible 5x candidates.

## Ranking Fields

For each stock, score:

- `Regime Fit`
- `Theme Strength`
- `Leader Quality`
- `Catalyst Strength`
- `Market-Cap Fit`
- `Technical Setup`
- `Risk/Reward`

## Ranking Rules

- Rank by forward setup quality, not past performance alone.
- Penalize names that are too large for clean 5x math.
- Penalize names where the only thesis is crowding or low float.
- Reward names where business, catalyst, and chart are aligned.
- When two names are close, prefer the one with clearer invalidation and better liquidity.
