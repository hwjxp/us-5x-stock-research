# 中文评分卡

适用场景：

- 看单只股票有没有成为 `5x` 候选的潜力
- 对一批候选股票做横向比较
- 把主观感觉压缩成结构化判断

## 使用方法

每项打 `1-5` 分，满分 `35` 分。

- `1` = 很差
- `2` = 偏弱
- `3` = 中性
- `4` = 较强
- `5` = 非常强

## 七维评分

### 1. 宏观环境匹配度

看什么：

- 当前市场是否愿意给成长、高弹性、主题股估值
- 利率、流动性、风险偏好是否友好
- 这一类股票最近是否有板块效应

打分指引：

- `5`：明显顺风，板块和风格都在强化
- `3`：中性，可做但不是明显加分项
- `1`：明显逆风，容易被估值压制

### 2. 主题强度

看什么：

- 赛道是不是大方向
- 未来 `1-3` 年空间够不够大
- 主题是长期趋势还是短期口号

打分指引：

- `5`：大叙事、大空间、持续有资金关注
- `3`：有故事，但持续性一般
- `1`：小题材，缺乏延展性

### 3. 龙头质量

看什么：

- 是不是龙头或强二
- 产品、渠道、成本、品牌、网络效应是否领先
- 是否具备机构资金能接受的质量

打分指引：

- `5`：行业核心资产，领先优势清晰
- `3`：有竞争力，但不是最优载体
- `1`：边缘玩家，容易被替代

### 4. 催化强度

看什么：

- 未来 `3-12` 个月有没有能让市场重估的事件
- 财报、订单、产品、临床、监管、降息、行业价格拐点是否明确

打分指引：

- `5`：催化明确、时间窗清晰、兑现后影响大
- `3`：有潜在催化，但模糊
- `1`：基本靠想象，没有硬触发器

### 5. 市值与流通盘匹配度

看什么：

- 现有市值是否还允许 `5x`
- 自由流通盘是否合适
- 平均成交额能否支撑趋势

打分指引：

- `5`：市值不大不小，空间和流动性兼备
- `3`：要么太小风险大，要么稍大但仍有空间
- `1`：要么太大难 `5x`，要么太小太脏

### 6. 技术形态质量

看什么：

- 是否收复 `50DMA` 和 `200DMA`
- 是否刚突破大平台
- 是否放量
- 是否具备沿 `20DMA/50DMA` 走趋势的能力

打分指引：

- `5`：典型主升浪起点
- `3`：结构一般，还需等待
- `1`：图形破碎，不适合做趋势

### 7. 风险收益比

看什么：

- 上行空间和下行风险是否对称
- 失败后会不会快速归零或大幅稀释
- 当前追进去是不是已经太晚

打分指引：

- `5`：赔率明显好于风险
- `3`：有机会，但容错一般
- `1`：上行空间有限，下行风险很大

## 总分解释

- `28-35`：重点研究区，值得进入核心观察名单
- `22-27`：有机会，但还需要等催化或等图形更好
- `15-21`：偏观察，不适合重仓
- `<15`：不符合这套 `5x` 研究框架

```text

```
