# US 5x Stock Research

A Codex skill for studying U.S. stocks through a `5x-10x` multi-bagger framework.

## Included

- `SKILL.md`: main skill entry
- `agents/openai.yaml`: UI metadata
- `references/`: framework, technical setups, scorecards, and execution playbooks

## Focus

- regime analysis
- theme selection
- leader identification
- catalyst mapping
- market-cap and float fit
- technical entry and exit logic
- position sizing and trade management

## Main Skill Folder

This repository is structured as a standalone skill package. The core files are ready to be copied into a local Codex skills directory or versioned as a reusable research skill.
